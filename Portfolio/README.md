# Aarav Shah — Systems in Motion

A responsive, interactive personal portfolio exploring robotics, quantitative finance, music, engineering, and leadership through one central idea: **Different systems. One mindset: disciplined improvement.**

## Live website

[aaravshahportfolio.vercel.app](https://aaravshahportfolio.vercel.app)

The Vercel project should use `Portfolio/` as its Root Directory. This repository keeps the app in that folder so the portfolio source and its deployment configuration stay together.

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm test
npm run build
npm run lint
```
